Angular CLI 21 and Mysql starter project

Dependencies

1. Node 24.13.1
2. Npm 11.8.0

Features:

1. GraphQL integration from Front-End
2. Custom Login, Logout and Account Registration
3. Multi-Factor Authenticator- MFA
4. Users' Profile, Change Password, Disable/Enable MFA and Upload User Profile Picture
5. Products List, Catalogs Custom Pagination and Product Search
6. Product PDF Report
7. Sales Graph
